# Design Anti-Defaults

Reject unless explicitly requested:

- Generic blue/purple SaaS gradients
- Hero headline plus three generic cards
- Endless floating card sections
- Decorative blobs/orbs as the main visual idea
- Beige wellness default without a brand reason
- Dark dashboard with no domain signal
- Stock-looking blurred hero imagery
- Icons used only as decoration
- Placeholder copy that explains the UI instead of usable content
- One-note palette
- Desktop polish with cramped mobile
