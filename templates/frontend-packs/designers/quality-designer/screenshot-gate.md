# Screenshot Gate

Required viewports:

- Mobile: 390 x 844
- Desktop: 1440 x 1000

Optional:

- Narrow mobile: 320 x 740
- Wide desktop: 1728 x 1117

Check:

- first viewport
- primary workflow
- dense data state
- empty state
- loading state
- longest expected copy

Revise before final if screenshots expose layout, hierarchy, contrast, or domain-fit problems.
